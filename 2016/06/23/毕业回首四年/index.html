毕业回首四年

发表于 2016-06-23   |   分类于 生活   |     |   阅读次数

前言

最近看了一次 TED 的演讲，也是思考自我到底是什么（我从未接触过哲学，这里只以最质朴的眼光来看问题 :]），最通俗的看法自我就是承载着经历和回忆的这具身体，而其提出的看法认为经历与回忆等等和身体实体共同构成了自我，身体实体并不比感觉起来虚幻一些的经历和回忆更重要。所以在我来看，这四年并不只是我经历过，而是已经成为了我生命的一部分，融入了我身上拖负着的世界里去。

很久没有更新博客了，最近一直在疯玩忙着毕业没摸技术也没空思考，反倒是每次在火车上的那十几个小时总能收获高质量的阅读和感悟。其实我本质上也只是个自控力一般的三分钟热度型人，内里的懒惰和随性一直在与周遭环境给的压力和对技术的兴趣作着扩日持久的斗争，好习惯们也在不断地尝试培养和难以坚持中此消彼长，唯有对于未来的热切希冀如一直悬在头顶的达摩克利斯之剑保证我在短暂的沉溺之后继续向前挣扎。

大学四年

相比与初高中平淡舒适的学习和生活，大学给了我各种各样的第一次体验。犹记得刚到时和康总在交大东路闲逛的迷茫，第一次离家这么远，第一次住集体宿舍，第一次去公共澡堂洗澡，第一次刷夜，第一次参加社团，第一次自己独立生活。大多数事情船到桥头自然直了，还有一些慢慢地适应了不适应也得适应，总的来说确实比高中累多了但是学到的东西也多多了。学习方面倒是一如既往地尽自己努力然后听天命，生活方面从一开始的略洁癖到后来也差不多被室友治好了，人际方面还是不太擅长也不太喜欢只是真心地对待自己小圈子里的每一个人，技术方面跟着几位室友大神走不被拉得太远我也很满足了。

大一刚来时和室友还比较生疏，因为生活习惯也有一些小摩擦，投入时间更多的是社团的事情。期间在青团帮志愿者录制有声图书送给盲人社区，每周定期去食堂和教学楼收集失物统一进行招领；在阳服大家齐心协力为同学们向信息中心争取每个月更多的流量，又接着向后勤部门争取延后澡堂关门时间；作为学生会的一员还被拉着去给梦想合唱团第二季开播盛典当观众（要到了小撒的签名可是忘了合影 :[），去北京国际马拉松当志愿者在天安门维持秩序。这些都是以往不曾经历过或者只有在电视上才能看见的场景，有幸在北京这个充满奇迹的地方一一体验过了。不过我得承认即使在经历过一些有意义的事情后，我还是不太喜欢学生会那种条条框框效率低下的工作模式，所以大一之后我就果断退掉了所有社团。

大二大三的生活就比较平淡以课业为主了，要么就是开学初死命抱住我的室友大腿们，要么只能接受每个学期快结束前的那几周的煎熬，不是在与组员无休止的开会协调中就是在一个人拖着几个甩不掉的酱油默默地玩命写代码。然后就是担心自己没有保研的成绩也肯定自己没有考研的想法，企图仗着自己还可以的英语成绩强行考出雅思成绩出国读研，最后总分够了却还是死在了口语和写作上，权衡利弊之后还是果断放弃了这条路。又回头参加大创和其他活动企图争取到哪怕一点点加分，可是相比于其他参加了各种工作拿了各种荣誉的同学还是杯水车薪。大概算了一下自己应该就是处于保研边缘那个位置，还受到学员争取的保研名额以及其他同学各种因素影响，然后大三快结束一直到大四开学的这段时间就是在惴惴不安中度过。

大四算是幸运的一年，刚回到家就被北大夏令营召回，面试连时间复杂度都答错了竟然还给过了，之后为了准备下学期可能的出国交换机会赶紧找了实习，开学之后保研结果也终于下来靠着实打实的成绩和各种运气也是保上了。实习期间没事就溜到公司边上大厦里的留学服务中心办手续，实习刚结束加拿大的留学之旅也顺利成行，在那边也是一边忙着生存忙着课程一边准备着国内的毕业设计，留学回国没多久就是毕业答辩，论文最后一点还是我在给我哥当伴郎的路途中完成的，最后也顺利毕业了！

结语

这四年过的拥挤却热闹，躁动后也思考，散训月考大项目经历不少，开开心心一起丢节操。有傲娇班委，有超神学霸，有技术大牛，有闷骚逗比，我们聚在运动场／召唤师峡谷／天佑舞台上，我们散落到各自的回忆里去，犹记得那一声充满感情的召唤：

骚年，晚上撸串儿去呀～
